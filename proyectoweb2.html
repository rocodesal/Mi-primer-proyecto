En los diferentes formularios utilicé las etiquetas legend para poder incluir la leyenda "datos personales" en el fieldset. Incluí la etiqueta file en el input para que el usuario pudiera seleccionar una imagen en este caso o un archivo en situaciones generales.

Por ejemplo, en el primer formulario ingresé la etiqueta submit para agregar el botón de enviar que permita poder enviar el mismo. Sin embargo en el siguiente, además de haber incluido la eqtiqueta maxlength para limitar el número de caracteres a ingresar por parte del usuario, también agregué un botón de "enviar", pero con la etiqueta "button".

En tanto, también utilicé el atributo "textarea", para delimitar un área donde el usuario pudiera dejar su comentario. EL área del atributo, fue delineado de forma razonable con los valores cols y rows.

Siendo VIernes 01:52, acabo de applicar el atributo "label" para colocar el valor nombre fuera del recuadro y así hacer más sencillo el ingreso de los datos al usuario, ya que, con el valor "type" y "value", el usuario debía borrar el texto para poder ingresar sus datos. Seguramnte deba haber alguna forma de que el texto preliminar colocado dentro del recuadro desaparezca al hacer click, pero aún no lo he visto en el curso actual.
